
# NHS Health Inequalities Analysis

## Project Overview

This project analyses NHS Healthy Life Expectancy (HLE) data to investigate health inequalities across deprivation levels, demographic groups, and time periods.

The project combines Python for data analysis and Power BI for dashboard development to generate actionable public health insights.

## Objectives

- Analyse Healthy Life Expectancy trends
- Measure deprivation-related health inequalities
- Compare outcomes across demographic groups
- Identify long-term inequality trends
- Develop an interactive Power BI dashboard

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- SciPy
- Power BI

## Key Findings

- Healthy Life Expectancy increases substantially as deprivation decreases.
- The deprivation gap reached approximately 12.36 years.
- Females showed slightly higher Healthy Life Expectancy than males.
- Health inequalities widened in recent years.
- Socioeconomic deprivation remains the strongest driver of inequality.
